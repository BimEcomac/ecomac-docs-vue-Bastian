## **Batman**
**Batman es el nombre del servidor donde esta alojado Ecoline. Este servidor proporciona la infraestructura necesaria para garantizar el funcionamiento y la disponibilidad continua de Ecoline, asegurando un rendimiento óptimo y una alta fiabilidad.**
<Aside type="note" title="Mantis">***Mantis es una base de datos utilizada en el servidor llamado Batman. Este servidor alberga diversas aplicaciones o servicios que dependen de Mantis para almacenar y gestionar sus datos***</Aside>
### *Ecoline*
- Ecoline es el CRM utilizado por Ecomac, diseñado para gestionar y optimizar las relaciones con los clientes. Este sistema ayuda a Ecomac a manejar las interacciones y datos de los clientes, mejorar el servicio al cliente, y aumentar la eficiencia operativa. Ecoline permite a Ecomac realizar un seguimiento de las ventas, gestionar el soporte al cliente, y analizar datos para tomar decisiones informadas que impulsan el crecimiento del negocio.
