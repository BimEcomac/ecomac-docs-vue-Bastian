## Primeros pasos

Vagrant lo necesitarás para crear tu propio entorno de desarrollo en conjunto a Virtual Box.

¿Para qué sirve Vagrant? 

Vagrant aísla las dependencias y su configuración dentro de un único entorno desechable y consistente, sin sacrificar ninguna de sus herramientas existentes.

## Instalación de Vagrant

Puedes instalar Vagran de dos maneras, mediante comandos en la consola de mondos o descargando el installador desde la página oficial en la cual tambien esta la documentación que veras a continuación. 

<Card title="" icon="">
### Linux y macOS 
Para una instalación mediante la consola de comandos deveras utilizar lo siguiente:

<Tabs>

  <TabItem label="macOS" icon="apple">
    ```bash
    brew tap hashicorp/tap
    brew install hashicorp/tap/hashicorp-vagrant
    ```
  </TabItem>
  <TabItem label="Linux" icon="linux">
    ```bash
    wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
    echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
    sudo apt update && sudo apt install vagrant
    ```
  </TabItem>
</Tabs>
</Card>
