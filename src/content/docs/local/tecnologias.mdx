### Corramos el ambiente!
Una vez ya estando en el repositorio del proyecto **ecomac-docs** tenemos que proceder a instalar
los packetes de Node para poder levantar nuestro ambiente. 

    **Comado en git bash**
```bash
 npm install
```
Y una vez se hayan descargado he instalado todos los paquetes, tenemos que iniciar nuestro proyecto con el siguiente comando
```bash
 npm run dev
```
<Aside type="note" title="Nota">Puedes utilizar la consola de comandos en visual studio utilizando el siguiente atajo de teclado: **ctrl + ñ** </Aside>

### Función de comandos

| *Comandos*	| *Ación*    |
|-----------|-----------|
|   **npm install** |   Instala las dependencias   |
|   **npm run dev**	|   Inicia un server local de desarrollo en **localhost:4321**  |
|   **npm run build**   |  	Construye tú sitio de producción para **./dist/**   |
|   **npm run preview** |	Obtenga una vista previa de su compilación localmente, antes de implementarla    |
|   **npm run astro ...**   |	Ejecute comandos CLI como **astro add**, **astro check**    |
|   **npm run astro -- --help** |	Obtenga ayuda para usar Astro CLI    |
|   **git clone [url]** | Comando de git para clonar el proyecto    |
|   **git fetch**   |   Comando de git para obtener la informacion actual del proyecto   |
|   **git pull**   |   Comando de git para traer todos los cambios de la rama a tú repositorio |
